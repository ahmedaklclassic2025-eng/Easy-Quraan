## 🔐 خطوات النشر

### 1. توقيع التطبيق

قبل رفع التطبيق إلى Google Play، يجب توقيعه:

```bash
# إنشاء keystore (مرة واحدة فقط)
keytool -genkey -v -keystore easy-quraan-key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias easy-quraan

# توقيع APK
jarsigner -verbose -sigalg SHA256withRSA -digestalg SHA-256 -keystore easy-quraan-key.jks المصحف-الميسر-v1.7-release.apk easy-quraan

# تحسين APK (اختياري لكن موصى به)
zipalign -v 4 المصحف-الميسر-v1.7-release.apk المصحف-الميسر-v1.7-release-aligned.apk
```

**مهم:** احتفظ بملف keystore في مكان آمن! ستحتاجه لكل تحديث مستقبلي.

### 2. إنشاء حساب Google Play Console

1. اذهب إلى https://play.google.com/console
2. أنشئ حساب مطور (رسوم لمرة واحدة: $25)
3. املأ معلومات الحساب

### 3. إنشاء تطبيق جديد

1. اضغط على "Create app"
2. املأ المعلومات الأساسية:
   - اسم التطبيق: المصحف الميسر
   - اللغة الافتراضية: العربية
   - نوع التطبيق: App
   - مجاني/مدفوع: مجاني
